# 프로젝트 2 · 자유 주제 자동화 (날씨 알림)

> 작업 예정. 아래는 계획 요약입니다.

## 반복 업무 정의

매일 아침 날씨를 확인하고, 비 예보가 있으면 우산을 챙기라고 알려주는 작업을 자동화한다.

## 워크플로우 (예정)

```
[Trigger]  매일 아침 정해진 시각 (스케줄)
      │
[Action]   날씨 API 조회
      │
[조건 분기]  강수확률 기준 (예: 60% 이상 / 미만)
      │
      ├─ 비 예보 → Discord "우산 챙기세요" 알림
      └─ 맑음    → Discord 일반 브리핑 알림
```
